# ipa 包怎么安装？

因为这是没签名的 ios 应用，所以需要手动安装。
ipa 包安装方式有多种：

1. 越狱的手机通过巨魔商店就可以安装，简单方便。
2. 未越狱的手机，可以试试爱思助手安装 IPA：[https://www.i4.cn/news_detail_38195.html](https://www.i4.cn/news_detail_38195.html)，个人苹果 ID 自签有效期 7 天，可重复续签，也可使用 SideStore 或 LiveContainer 或者 AltStore 等工具自签名，B 站都有视频教程。
3. 未越狱的手机也可在淘宝 14.5 买一个一年的签名文件，配合万能签或者轻松签或全能签等工具，自签任意 ipa 文件，安装好就可以使用一年，简单方便。

# 不越狱长期安装 ipa

如果你想长期使用，并且手机不想越狱，可以使用签名工具，来给你这个应用签名，然后安装。当前签名工具需要收费，淘宝搜一下：ipa 签名，就可以找到。常见的万能签，全能签，轻松签等等，都可以。一年也就 15 元左右的费用，但是功能是相当强大的，可以签任何 ipa 包，然后安装到自己的手机上。按照人家的使用教程来，就可以了。

# apk 包怎么安装？

安卓手机直接将 apk 包发送到手机上，应该就可以安装了，华为可以使用卓易通还是啥工具也可以安装 apk 安装包。
